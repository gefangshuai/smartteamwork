smart Teamwork 团队协作系统
---------
##计划包含功能：
* 项目信息管理
* 开发组管理
* 开发组分组讨论
* 个人任务TODO管理
* TODO进度跟踪
* 集成第三方消息（后期会考虑）

##使用技术
* JFinal轻量级mvc框架及JFinal-ext扩展
* Shiro权限控制
* jsp 2.0标签布局
* bootstrap 3.0前端框架
